# Sequencer Subtitles

**Sequencer Subtitles** is a subtitle display plugin for Unreal Engine 5.
Add a dedicated track to the Sequencer timeline and place subtitles with real-time editor preview.

> **Download from Fab**: [Sequencer Subtitles on Fab](https://www.fab.com/listings/a5ee3c5a-6aee-4224-841e-2c68880325c8)

## Features

- Adds a **Subtitle Track** to Sequencer
- Real-time preview in the editor
- Customizable UI via Widget Blueprint

## Installation

1. Download the zip matching your UE version from the [Releases](https://github.com/kokagefujieda/SequencerSubtitles/releases) page
2. Copy the `SequencerSubtitles` folder into your project's `Plugins/` directory
3. Launch the UE editor and enable the plugin
4. Open Sequencer and add via `+ Track` → **Subtitle Track**
