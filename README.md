# Sequencer Subtitles

**Sequencer Subtitles** は Unreal Engine 5 向けの字幕・台詞表示プラグインです。
Sequencer タイムラインに専用トラックを追加し、リアルタイムプレビュー付きで字幕を配置できます。

## 特徴

- Sequencer に **Dialogue Track** を追加
- エディタ上でのリアルタイムプレビュー
- Widget Blueprint でカスタマイズ可能な UI
- Fab にて配布中

## 動作環境

- Unreal Engine 5.5 / 5.6 / 5.7

## インストール

1. `Plugins/SequencerSubtitles` フォルダをプロジェクトの `Plugins/` にコピー
2. UE エディタを起動し、プラグインを有効化
3. Sequencer を開き、`+ Track` → **Dialogue Track** を追加
